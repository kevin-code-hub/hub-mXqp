
## 5. 模型结构设计

### 5.1 Siamese Network 总体结构

```
Input Sentence
      ↓
Sentence Encoder（共享参数）
      ↓
Sentence Embedding (B, H)
      ↓
Triplet Loss / Cosine Similarity
```

同一个 Encoder 被复用 3 次：

* a / p / n 共用权重

---

### 5.2 Sentence Encoder（LSTM）

当前使用 **BiLSTM + Pooling**：

```text
Embedding → BiLSTM → Linear → MaxPooling → Sentence Vector
```

关键点：

* **双向 LSTM** 捕获上下文信息
* `Linear(H*2 → H)` 压缩维度
* `Max Pooling` 得到定长句向量

输出：

```text
(Batch, HiddenSize)
```


### 5.2 使用 BERT

只需替换 **Sentence Encoder**：

```text
BERT → CLS / Pooling → Sentence Vector
```


---

### 5.3 损失函数

默认使用 PyTorch 内置：

```python
nn.TripletMarginLoss(margin=0.1, p=2)
```

目标：

```text
d(a, p) + margin < d(a, n)
```

使：

* 同语义问题更接近
* 不同语义问题更远

## 6. 评估方式设计

### 6.1 知识库向量化

评估前：

1. 使用训练集构建「问题向量库」
2. 每个问题 → encoder → embedding
3. 所有向量进行 L2 归一化

---

### 6.2 测试流程

对每个测试问题：

1. 编码为向量
2. 与知识库中所有问题计算 cosine similarity
3. 取最大相似度对应的问题
4. 判断其标准问是否与真实 label 一致

评价指标：

```text
Accuracy = correct / (correct + wrong)
```

## 7. 训练流程与实验设计

* optimizer：Adam
* learning rate：多组对比（1e-3 ~ 5e-4）
* batch size：64 / 128
* epoch：15

每轮：

1. Triplet 训练
2. 验证集评估
3. 记录 best acc / epoch / time
