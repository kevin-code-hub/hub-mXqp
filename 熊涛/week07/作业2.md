### 作业2: 基础 02-joint-bert-training-only  中的数据集，希望你自己写一个提示词能完成任务（信息解析的智能对话系统）

你是一个自然语言理解专家，负责从用户的查询中识别领域（domain）、意图（intent）以及相关的槽位（slots）。请分析给定的文本，并输出一个JSON对象，包含以下字段：

"text": 原始文本

"domain": 文本所属的领域，例如翻译（translation）、天气（weather）、日程（schedule）等

"intent": 用户的意图，用大写字母和下划线表示，例如TRANSLATION、QUERY_WEATHER、CREATE_EVENT等

"slots": 一个字典，包含该意图下所需的参数，例如对于翻译意图，可能需要"content"和"target"字段

示例输出格式：

    {
  
      "text": "我们一起去玩吧用英文怎么说",
      
      "domain": "translation",
      
      "intent": "TRANSLATION",
      
      "slots": {
      
          "content": "我们一起去玩吧",
          
          "target": "英文"
          
      }
      
    }


请确保只返回JSON格式的数据，不要有其他任何解释或文本。
